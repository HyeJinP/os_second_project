START

It is easy to make webpage with ERICAST!

● Make sure you're connected to the Internet. Other pages, including this page, may not work properly if you are not connected to the Internet.

● Enable the 'CREATE' feature. (It is initially disabled. See working version at http://www.sbnation.com/a/meme.) To enable this, additional installation is required. Please read the README.md file in the 'meme' folder for further instructions.

● Take a look at ERICAST and edit the code as you wish. See the 'FUNCTION' section for usage.

That's all you have to do!

If you get stuck, contact us! We will gladly help you.
